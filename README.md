# Odoo 9
Instalação do Odoo 9.0 no Ubuntu 14.04


Baixar o script
> sudo git clone https://github.com/bilibidum/Odoo9.0.git

Entrar no diretório
> cd Odoo9.0

Dar permissão para execução
> sudo chmod +x odoo_install.sh

Executar o script
> sudo sh odoo_install.sh
